## 📥 Installation Guide

1. **Mod Installation (Fabric 1.21.1):** Move the mods from the "Mods" folder into your Minecraft Fabric 1.21.1 instance folder.

2. **Client Setup:** Run `installer.exe`. If the .NET 8.0 installation fails, download it manually here: [Official .NET 8.0 SDK](https://dotnet.microsoft.com/en-us/download/dotnet/thank-you/sdk-8.0.421-windows-x64-installer).

3. **Activation & Usage:** Launch FanaticClient and enter your key. If you encounter an "HWID already linked" error, open a ticket on our [Discord](https://discord.gg/Wr3mYUGtJN) for a reset.

4. **Getting Started:** Once in-game, wait for the red icon to turn green to start using the modules.
